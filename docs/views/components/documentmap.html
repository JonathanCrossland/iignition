Document Map Component

A component that displays a grid of document pages with configurable layout, loading states, and interactive features.

Basic Document Map

Document Map with Titles

Custom Styled Document Map

 
<!-- Basic Document Map -->
<document-map></document-map>

<!-- Document Map with Titles -->
<document-map show-title></document-map>

<!-- Custom Styled Document Map -->
<document-map class="custom-map"></document-map>
Property Type Default Description
show-title boolean false Whether to display page titles
Method Parameters Description
configure config: Partial<DocumentMapConfig> Configures the document map with custom settings
loadData xmlDoc: XMLDocument Loads and displays document data from XML
Property Type Default Description
pageConfig.selector string 'page' Selector for page elements in XML
pageConfig.titleSelector string 'chapter' Selector for page titles in XML
pageConfig.contentSelector string 'paragraph' Selector for page content in XML
onPageClick function undefined Callback when a page is clicked
