Splash

Splash is a utility class that allows you to map data to a template and apply it to an element.

The Splash component provides data binding functionality through the data-splash attribute. It can:

  • Map data to HTML elements using attribute-based binding
  • Handle template cloning for repeated elements
  • Support custom row binding through callbacks
  • Automatically handle different input types (text, checkbox, etc.)

Data Binding

Use the data-splash attribute to bind data to elements:

<div data-splash="name"></div>
<input type="text" data-splash="email">
<input type="checkbox" data-splash="isActive">

Template Usage

For repeated elements, use data-splashtemplate:

<div id="users">
    <div data-splashtemplate>
        <span data-splash="name"></span>
        <span data-splash="email"></span>
    </div>
</div>

<script>
    const data = [
        { name: 'John', email: 'john@example.com' },
        { name: 'Jane', email: 'jane@example.com' }
    ];
    
    $i.splash.map('#users', data);
</script>

Custom Row Binding

You can provide a custom binding function for each row:

$i.splash.map('#users', data, (element, item) => {
    // Custom binding logic
    element.classList.add(item.isActive ? 'active' : 'inactive');
});

Attribute Binding

Bind data to specific attributes using the format attribute=property:

<img data-splash="src=imageUrl">
<a data-splash="href=profileUrl">Profile</a>
Element Type Binding Behavior
input[type="text"] Sets the value property
input[type="checkbox"] Sets the checked property based on boolean or numeric value
textarea Sets the value property
Other elements Sets the textContent

Data Splashed Event

Listen for when data has been applied:

$i.on('onDataSplashed', (event) => {
    console.log('Data splashed:', event.detail);
});
