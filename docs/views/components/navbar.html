Nav Bar Component

A compact, fixed-width vertical navigation bar with icon buttons, collapsible groups, and custom tooltips styled like VSCode. The navigation bar consists of three components: nav-bar (container), nav-group (collapsible group), and nav-item (individual icon button).

Custom Tooltip: When you hover over a nav-item or nav-group, a custom tooltip appears to the right of the nav bar, styled like VSCode. This tooltip uses the tooltip or label attribute value.

nav-bar

The main container component that organizes navigation items into top and bottom sections.

nav-item

Individual icon button that can be clicked to trigger navigation. Place SVG elements or other content directly inside.

nav-group

Collapsible group that can contain multiple nav-items, similar to a folder in a file explorer. Use slot="icon" for placing SVG icons in the group header.
  • The header area appears at the top of the nav bar, above the navigation items.
  • Use the slot="header" attribute to place content in the header.
  • You can add a logo, text, or any other content in the header slot.
  • Add a nav-header-separator div to create a separator line below your header content.
<nav-bar>
    <div slot="header">
        <svg width="32" height="32" viewBox="0 0 24 24" style="margin: 8px 0 12px">
            <path fill="#ff6b00" d="M10 20v-6h4v6h5v-8h3L12 3 2 12h3v8z"/>
        </svg>
        <div class="nav-header-separator"></div>
    </div>
    <!-- nav items here -->
</nav-bar>

nav-item Properties

Property Type Description
tooltip string Text to display in the custom tooltip when hovering
label string Alternative to tooltip, used as fallback if tooltip is not provided
active boolean Whether the item is currently active/selected
dock string Set to "bottom" to place the item at the bottom of the nav bar

nav-group Properties

Property Type Description
label string Text to display in the tooltip when hovering over the group header
collapsed boolean Whether the group is initially collapsed
dock string Set to "bottom" to place the group at the bottom of the nav bar

nav-item Events

Event Name Detail Description
nav-item-click { id, label, tooltip } Emitted when a navigation item is clicked

nav-group Events

Event Name Detail Description
nav-group-toggle { collapsed } Emitted when a navigation group is expanded or collapsed
Interact with the nav bar to see events
const navbar = document.querySelector('#eventExample');
navbar.addEventListener('nav-item-click', (e) => {
    console.log('Nav item clicked:', e.detail);
});
navbar.addEventListener('nav-group-toggle', (e) => {
    console.log('Nav group toggled:', e.detail);
});
