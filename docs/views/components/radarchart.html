Radar Chart

A customizable radar chart component that visualizes multi-dimensional data in a radial format, perfect for displaying metrics, skills, or comparative analysis.

Basic Radar Chart

Custom Styled Radar Chart

The Radar Chart component can be used to visualize multi-dimensional data on a radial grid. Each axis represents a different metric, and the data points are connected to form a polygon.

<!-- Basic Radar Chart -->
<radar-chart data="-0.5,1,0.5,0.8,0.3,0.9,0.7"></radar-chart>

<!-- Custom Styled Radar Chart -->
<radar-chart 
    data="0.8,0.6,0.9,0.7,0.5,0.4,0.8"
    style="
        --grid-line-color: #444;
        --axis-line-color: #666;
        --data-line-color: #ff6b00;
        --data-fill-color: rgba(255, 107, 0, 0.2);
    ">
</radar-chart>
Property Type Description
data string Comma-separated values between -1 and 1 representing the data points for each axis

Default Labels

The component comes with default labels that can be customized through the component's code:

  • Agile
  • Minimal
  • Maintainable
  • Environmental
  • Reachable
  • Solvable
  • Extensible

The Radar Chart component can be customized using CSS variables:

CSS Variable Default Description
--radar-bg #fff Background color of the chart
--radar-grid #ccc Color of the grid lines
--radar-axis #000 Color of the axis lines
--radar-label #000 Color of the axis labels
--radar-line green Color of the data polygon outline
--radar-fill rgba(0, 255, 0, 0.5) Fill color of the data polygon

Example Styling

:root {
    --radar-bg: #1e1e1e;
    --radar-grid: #444;
    --radar-axis: #666;
    --radar-label: #fff;
    --radar-line: #ff6b00;
    --radar-fill: rgba(255, 107, 0, 0.2);
}
