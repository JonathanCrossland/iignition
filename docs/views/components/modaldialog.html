Modal Dialog Component

A flexible modal dialog component that can be used to display content, forms, or confirmations.

This is a modal dialog.

 
<modal-dialog visible="true" buttons="ok,cancel">
    <p>Your content here</p>
</modal-dialog>
Property Type Description
visible boolean Controls the visibility of the modal
buttons string Comma-separated list of buttons to show (ok, cancel, close)
title string Title displayed in the modal header
CSS Variable Default Description
--modal-bg #252526 Background color of the modal
--modal-color #cccccc Text color of the modal content
--modal-border-radius 8px Border radius of the modal
--modal-padding 1em Padding inside the modal
--modal-shadow 0 4px 6px rgba(0, 0, 0, 0.3) Box shadow of the modal
--modal-overlay-bg rgba(0, 0, 0, 0.4) Background color of the overlay
--modal-button-bg #0098fd Background color of primary buttons
--modal-button-color #ffffff Text color of primary buttons
--modal-button-hover-bg #0098fd Background color of primary buttons on hover
--modal-button-cancel-bg #313131 Background color of cancel buttons
--modal-button-cancel-hover-bg #464646 Background color of cancel buttons on hover
--modal-title-color #9b9b9b Color of the modal title
--modal-close-color #cccccc Color of the close button
--modal-close-hover-color #0098fd Color of the close button on hover
--modal-header-bg #363636 Background color of the modal header
--modal-button-border-color transparent Border color of primary buttons
--modal-button-hover-border-color transparent Border color of primary buttons on hover
--modal-button-cancel-border-color transparent Border color of cancel buttons
--modal-button-cancel-hover-border-color transparent Border color of cancel buttons on hover
