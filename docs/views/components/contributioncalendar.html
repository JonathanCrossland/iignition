A GitHub-style contribution calendar component. Displays a grid of cells representing activity levels (e.g., word count) per day, with customizable color levels and tooltips for each cell.

Basic Usage

<contribution-calendar></contribution-calendar>

Loading Data

const calendar = document.querySelector('contribution-calendar');
calendar.setData([
    { year: 2024, month: 3, dayOfWeek: 1, level: 2 },
    { year: 2024, month: 3, dayOfWeek: 2, level: 4 },
    // ...
]);

Static Data via Attribute

<contribution-calendar data='[{"year":2024,"month":3,"dayOfWeek":1,"level":2}]'></contribution-calendar>

Data Format

Property Type Description
year number Year of the cell
month number Month (1-12)
dayOfWeek number Day of week (0=Sunday, 1=Monday, ... 6=Saturday)
level number Activity level (0-5, determines cell color)
Method Parameters Description
setData (data: Array<{ year: number, month: number, dayOfWeek: number, level: number }>) Sets the calendar data and re-renders the grid

The contribution calendar can be themed using CSS custom properties.

Property Default Description
--contribution-calendar-bg#161b22Background color of the calendar
--contribution-calendar-padding8pxPadding around the calendar
--contribution-calendar-radius6pxBorder radius of the calendar
--contribution-calendar-week-gap2pxGap between weeks (columns)
--contribution-calendar-cell-gap2pxGap between cells (rows)
--contribution-calendar-cell-size14pxSize of each cell
--contribution-calendar-cell-radius3pxBorder radius of each cell
--contribution-calendar-level-0#21262dColor for level 0 (no activity)
--contribution-calendar-level-1#0e4429Color for level 1
--contribution-calendar-level-2#006d32Color for level 2
--contribution-calendar-level-3#26a641Color for level 3
--contribution-calendar-level-4#39d353Color for level 4
--contribution-calendar-level-5#a6f6a6Color for level 5 (max activity)

Example Styling

contribution-calendar {
    --contribution-calendar-bg: #222;
    --contribution-calendar-level-5: #00ff00;
    --contribution-calendar-cell-size: 16px;
}
