Data Table Component

A dynamic table component that supports sorting, filtering, and pagination for displaying structured data.

Name Age Email
John Doe 30 john@example.com
Jane Smith 25 jane@example.com 
<data-table sortable>
    <table>
        <thead>
            <tr>
                <th data-sort="name">Name</th>
                <th data-sort="age">Age</th>
                <th data-sort="email">Email</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>John Doe</td>
                <td>30</td>
                <td>john@example.com</td>
            </tr>
        </tbody>
    </table>
</data-table>
Property Type Description
sortable boolean Enables sorting functionality for the table columns
data-sort string Specifies the column identifier for sorting (used on th elements)
CSS Variable Default Description
--table-bg #1e1e1e Background color of the table
--table-header-bg #252526 Background color of the table header
--table-border #3c3c3c Border color for table cells
--table-text #cccccc Text color for table content
--table-hover #2a2d2e Background color on row hover
