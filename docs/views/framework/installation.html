Installation

Get started with iignition by following these simple installation steps.

Required Files

To use iignition, you'll need to include two main JavaScript files in your project:

  • ii.js - The core framework
  • ii_components.js - The component library

HTML Setup

Include the files in your HTML document:

<!DOCTYPE html>
<html>
<head>
    <title>My iignition App</title>
    <!-- Include core framework -->
    <script src="path/to/ii.js"></script>
    <!-- Include components -->
    <script src="path/to/ii_components.js"></script>
</head>
<body>
    <!-- Your content here -->
</body>
</html>

File Structure

For optimal organization, we recommend the following structure:

your-project/
├── js/
│   ├── ii.js
│   └── ii_components.js
├── css/
│   └── styles.css
└── index.html

After Installation

Once you've included the required files, you can:

  1. Create view containers in your HTML
  2. Add components to your pages
  3. Set up navigation between views
