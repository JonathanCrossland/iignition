A file system explorer component that mimics VSCode's project explorer functionality. It provides an interactive tree view of files and folders with expand/collapse capabilities, selection states, and customizable icons.

The explorer component accepts a JSON structure representing your file system hierarchy and provides methods for interaction and state management.

Basic Usage

<explorer-component></explorer-component>

Loading Data

const explorer = document.querySelector('explorer-component');
explorer.load([
    {
        name: 'src',
        type: 'folder',
        expanded: true,
        children: [
            {
                name: 'components',
                type: 'folder',
                children: [
                    { name: 'app.ts', type: 'file' }
                ]
            }
        ]
    }
]);

Handling Selection

explorer.addEventListener('item-selected', (e) => {
    console.log('Selected item:', e.detail.path);
    console.log('Item type:', e.detail.type);
});

Change Detection

explorer.configure({
    onChange: (data) => {
        // Save updated structure
        console.log('Structure changed:', data);
    }
});

Data Structure

Property Type Description
name string Name of the file or folder
type 'file' | 'folder' Type of the item
expanded boolean? Whether the folder is expanded (folders only)
children ExplorerItem[]? Array of child items (folders only)

Methods

Method Parameters Description
load (data: ExplorerItem[]) Loads the file/folder structure into the explorer
configure (options: ExplorerOptions) Configures component options like onChange handler
getData () Returns the current data structure

Events

Event Detail Description
item-selected { path: string, type: string } Fired when an item is selected

The explorer component can be styled using CSS custom properties.

Property Default Description
--explorer-bg #1e1e1e Background color of the explorer
--explorer-text #cccccc Text color for items
--explorer-selected-bg #007fd4 Background color for selected items
--explorer-hover-bg rgba(255, 255, 255, 0.1) Background color for hovered items
--explorer-indent 20px Indentation for nested items
--explorer-item-height 22px Height of each item

Example Styling

explorer-component {
    --explorer-bg: #2a2a2a;
    --explorer-text: #ffffff;
    --explorer-selected-bg: #ff6b00;
    --explorer-hover-bg: rgba(255, 107, 0, 0.1);
    --explorer-indent: 24px;
    height: 400px;
}
