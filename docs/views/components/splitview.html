Split View Component

A component that creates resizable split views with customizable orientation and panel sizes.

left
right 
<split-view window-padding="0" orientation="row" style="height: 30vh; width: 100%;">
    <div slot="left" title="Outline" min-width="250px" justify-content="center">
        Left Panel Content
    </div>
    <div slot="right" title="Plotter">
        Right Panel Content
    </div>
</split-view>
Property Type Default Description
orientation string 'row' Orientation of the split view ('row' or 'column')
min-width string '100px' Minimum width for each panel
window-padding string '0' Padding for the window content
CSS Variable Default Description
--split-view-bg #1e1e1e Background color of the split view
--split-view-bg-light #252526 Lighter background color variant
--split-view-border #3c3c3c Border color
--split-view-text #cccccc Text color
--split-view-splitter rgb(54, 49, 49) Splitter background color
--split-view-splitter-hover #007fd4 Splitter hover color
--split-view-scrollbar-thumb #424242 Scrollbar thumb color
--split-view-scrollbar-thumb-hover #4f4f4f Scrollbar thumb hover color
--split-view-splitter-size 4px Splitter size
--split-view-splitter-hover-size 8px Splitter hover size
