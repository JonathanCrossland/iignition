Title Bar Component

A versatile and customizable title bar with application name, dropdown menu, and action buttons. The title bar consists of two components: title-bar (container) and title-action (individual action buttons).

You can style dropdown items using the following CSS classes:

.dropdown-item {
    padding: 8px 16px;
    cursor: pointer;
}

.dropdown-item:hover {
    background: rgba(255, 255, 255, 0.1);
}

.dropdown-separator {
    height: 1px;
    background: rgba(255, 255, 255, 0.1);
    margin: 6px 0;
}

Note: The Title Bar component is designed to fit the width of its container and maintain a fixed height. It handles text overflow with ellipsis to ensure it always fits properly in its container.

title-bar

The main container component that organizes the app name, dropdown menu, and action buttons.

title-action

Individual action button with optional label. Place SVG elements or other icon content directly inside.

title-bar Slots

Slot Name Description
app-name Content for the application name (left section)
dropdown-items Content for the dropdown menu items
actions Action buttons using title-action elements

title-bar Properties

Property Type Description
title string Text to display in the center dropdown button

title-action Properties

Property Type Description
label string Optional text label to display next to the icon

title-bar Events

Event Name Detail Description
dropdown-toggle { open } Emitted when the dropdown is opened or closed

title-action Events

Event Name Detail Description
title-action-click { id, label } Emitted when an action button is clicked
LitForge
Interact with the title bar to see events
const titlebar = document.querySelector('#eventExample');
titlebar.addEventListener('dropdown-toggle', (e) => {
    console.log('Dropdown toggled:', e.detail.open);
});
titlebar.addEventListener('title-action-click', (e) => {
    console.log('Action clicked:', e.detail);
});
CSS Variable Default Description
--titlebar-height 40px Height of the title bar
--titlebar-bg #1f1f1f Background color of the title bar
--titlebar-border #333 Border color at the bottom of the title bar
--titlebar-color #fff Text color in the title bar
--titlebar-font-size 14px Font size for text in the title bar
--titlebar-padding 0 16px Padding inside the title bar (must be specified on the title-bar element or parent scope)
--titlebar-dropdown-bg #252525 Background color of the dropdown menu
--titlebar-dropdown-hover-bg #333 Background color when hovering over dropdown button
--titlebar-dropdown-border #444 Border color of the dropdown menu
--titlebar-dropdown-shadow 0 4px 8px rgba(0,0,0,0.3) Box shadow of the dropdown menu
--titlebar-action-size 32px Size of the action buttons
--titlebar-action-spacing 8px Spacing between action buttons
--titlebar-action-hover-bg rgba(255,255,255,0.1) Background color when hovering over action buttons
--titlebar-action-icon-size 16px Size of the action button icons
--titlebar-action-label-size 12px Font size for action button labels
--titlebar-transition all 0.2s ease Transition duration and timing for animations

Example Custom Styling

/* Apply these variables either on :root, body, or directly on the title-bar element */
:root {
    --titlebar-height: 40px;
    --titlebar-bg: #1f1f1f;
    --titlebar-border: #333;
    --titlebar-color: #fff;
    --titlebar-font-size: 14px;
    --titlebar-padding: 0 16px; /* Important to set this to ensure proper layout */
    --titlebar-dropdown-bg: #252525;
    --titlebar-dropdown-hover-bg: #333;
    --titlebar-dropdown-border: #444;
    --titlebar-dropdown-shadow: 0 4px 8px rgba(0,0,0,0.3);
    --titlebar-action-size: 32px;
    --titlebar-action-spacing: 8px;
    --titlebar-action-hover-bg: rgba(255,255,255,0.1);
    --titlebar-transition: all 0.2s ease;
}

/* Or apply directly to the component itself */
title-bar {
    --titlebar-padding: 0 16px;
}
