A pill-shaped progress indicator that shows both target and current value percentages. The component features two overlapping bars with different heights and colors, percentage labels, and an optional title.

The progress-pill component can be used to visualize progress against a target. It consists of two overlapping bars - a base target bar and a taller value bar that shows the current progress.

Basic Usage

<progress-pill 
    title="Project Progress" 
    target="75" 
    value="60" 
    show-title="true">
</progress-pill>

With Custom Size

<progress-pill 
    title="Task Completion" 
    target="100" 
    value="85" 
    show-title="true" 
    size="80">
</progress-pill>
Property Type Default Description
title string '' The text displayed below the progress pill
target number 0 The target percentage (0-100)
value number 0 The current value percentage (0-100)
show-title boolean false Whether to display the title below the pill
size number 100 Scale factor for the entire component (100 = original size)

The progress-pill component can be styled using CSS custom properties. These properties can be set on the component itself or on a parent element.

Property Default Description
--progress-pill-height 120px Base height of the pill container
--progress-pill-target-color #0f0f0f Color of the target bar
--progress-pill-value-color #367900 Color of the value bar
--progress-pill-target-text-color #cccccc Color of the target label and title
--progress-pill-value-text-color #ffffff Color of the value label

Example Styling

progress-pill {
    --progress-pill-target-color: #2c3e50;
    --progress-pill-value-color: #27ae60;
    --progress-pill-target-text-color: #ffffff;
    --progress-pill-value-text-color: #ffffff;
}
  • Two overlapping progress bars with different heights
  • Percentage labels for both target and current value
  • Optional title display
  • Fully responsive to container width
  • Customizable colors through CSS variables
  • Smooth transitions for value changes
  • Scalable size through the size attribute
