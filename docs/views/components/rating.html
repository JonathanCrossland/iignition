Rating Component

A customizable rating component that allows users to display and collect ratings using stars or custom icons.

 
<rating-component 
    value="4"
    max="5"
    interactive
    star-icon="★"
    empty-star-icon="☆">
</rating-component>
Property Type Description
value number Current rating value
max number Maximum rating value (default: 5)
interactive boolean Whether the rating can be changed by user interaction
star-icon string Custom icon for filled stars
empty-star-icon string Custom icon for empty stars
CSS Variable Default Description
--rating-bg var(--color-card-bg) Background color of the rating component
--rating-star var(--color-warning) Color of filled stars
--rating-star-empty var(--color-border) Color of empty stars
--rating-text var(--color-text) Color of the rating text
--rating-size 24px Size of the stars
--rating-gap 4px Gap between stars
--rating-hover-scale 1.2 Scale factor on hover
--rating-transition 0.2s ease Transition animation
