Floating Button

A floating action button (FAB) is a circular button that floats above the UI and represents the primary action in an application.

Basic Floating Button

 
<!-- Basic Floating Button -->
<floating-button icon="add" position="bottom-right"></floating-button>

<!-- With custom icon -->
<floating-button vertical="bottom" horizontal="right" data-command="open-modal">
    <svg slot="icon" viewBox="0 0 24 24">
        <path d="M19 13h-6v6h-2v-6H5v-2h6V5h2v6h6v2z"/>
    </svg>
</floating-button>

<!-- Custom Styled Button -->
<floating-button class="custom-fab" icon="add" position="bottom-right"></floating-button>
Property Type Default Description
icon string add Icon to display on the button
position string bottom-right Position of the button (top-left, top-right, bottom-left, bottom-right)
vertical string bottom Vertical position (top, bottom)
horizontal string right Horizontal position (left, right)
data-command string undefined Command to execute when clicked
visible boolean true Controls the visibility and animation state
CSS Variable Default Description
--floating-button-size 56px Size of the floating button
--floating-button-radius 50% Border radius of the button
--floating-button-border none Border style of the button
--floating-button-bg var(--color-primary) Background color of the button
--floating-button-color var(--color-text-inverse) Color of the icon and text
--floating-button-shadow 0 2px 5px rgba(0,0,0,0.2) Box shadow of the button
--floating-button-transition 0.2s ease Transition duration and timing
--floating-button-hover-scale 1.05 Scale factor on hover
--floating-button-hover-shadow 0 4px 8px rgba(0,0,0,0.3) Box shadow on hover
--floating-button-hover-bg var(--color-primary-dark) Background color on hover
--floating-button-active-scale 0.95 Scale factor when clicked
--floating-button-icon-size 24px Size of the icon
--floating-button-margin 20px Margin from the edges
--floating-button-font-size 1rem Font size of any text content
