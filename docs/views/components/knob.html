Knob Component

A rotary control component that allows users to adjust values by rotating a knob, commonly used for volume controls, brightness adjustments, and other continuous value inputs.

 
<knob-component value="50" min="0" max="100"></knob-component>
Property Type Description
value number Current value of the knob
min number Minimum value of the knob
max number Maximum value of the knob
CSS Variable Default Description
--knob-size 100px Size of the knob component
--knob-track-width 8px Width of the track circle
--knob-track-color var(--color-border) Color of the track circle
--knob-progress-color var(--color-primary) Color of the progress indicator
--knob-handle-size 12px Size of the handle indicator
--knob-handle-color var(--color-text) Color of the handle indicator
--knob-text-color var(--color-text) Color of the value text
--knob-text-size var(--font-size-md) Size of the value text
