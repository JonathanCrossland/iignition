Tabbed View

A flexible tabbed interface component that supports dynamic content, sticky headers, and custom tab items.

The Tabbed View component consists of three main elements:

  • tabbed-view: The main container component
  • tab-panel: Individual content panels
  • tab-item: Optional items displayed in the tab header

Basic Structure

<tabbed-view>
    <tab-panel label="Tab 1">
        Content for first tab
    </tab-panel>
    <tab-panel label="Tab 2">
        Content for second tab
    </tab-panel>
</tabbed-view>

With Tab Items

<tabbed-view>
    <tab-panel label="Settings">
        Settings content
    </tab-panel>
    <tab-item>
        <button>Action Button</button>
    </tab-item>
</tabbed-view>

Tab Panel Properties

Property Type Description
label string The text displayed in the tab header
active boolean Indicates if the panel is currently active/visible

The Tabbed View component can be customized using CSS variables:

Variable Default Description
--tab-content-bg-color #ffffff Background color of the tab content area
--tab-inactive-bg-color #f5f5f5 Background color of inactive tabs
--tab-active-bg-color #ffffff Background color of the active tab
--tab-color #333333 Text color of the tabs
--tab-border-color #dddddd Border color for tabs
--tab-active-border-color #ff6b00 Border color for the active tab
--tab-padding 10px Padding for tab headers and items
--tab-font-size 9pt Font size for tab headers

Example Styling

tabbed-view {
    --tab-content-bg-color: #ffffff;
    --tab-inactive-bg-color: #f5f5f5;
    --tab-active-bg-color: #ffffff;
    --tab-color: #333333;
    --tab-border-color: #dddddd;
    --tab-active-border-color: #ff6b00;
    --tab-padding: 10px;
    --tab-font-size: 9pt;
}

  • Dynamic Content: Supports dynamic loading and updating of tab content
  • Custom Tab Items: Add custom elements to the tab header area
