Progress Bar Component

A progress bar component that displays the progress of a task or operation, with support for different colors and indeterminate state.

Basic Progress Bar

Indeterminate Progress Bar

Custom Progress Bar

 
<!-- Basic Progress Bar -->
<progress-bar value="75" max="100"></progress-bar>

<!-- Indeterminate Progress Bar -->
<progress-bar indeterminate></progress-bar>

<!-- Custom Progress Bar -->
<progress-bar value="30" max="100" class="custom-progress"></progress-bar>
Property Type Default Description
value number 0 Current progress value
max number 100 Maximum progress value
indeterminate boolean false Whether the progress bar is in indeterminate state
Method Parameters Description
setValue value: number Sets the current progress value
setMax max: number Sets the maximum progress value
setIndeterminate indeterminate: boolean Sets whether the progress bar is in indeterminate state
CSS Variable Default Description
--progress-height 4px Height of the progress bar
--progress-bg var(--color-border) Background color of the progress bar
--progress-color var(--color-primary) Color of the progress indicator
--progress-border-radius 2px Border radius of the progress bar
--progress-transition 0.2s ease Transition animation for progress changes
--progress-indeterminate-animation progress-indeterminate 2s linear infinite Animation for indeterminate state
--progress-text-color var(--color-text) Color of the progress text
--progress-text-size var(--font-size-sm) Size of the progress text
