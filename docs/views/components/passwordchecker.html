Password Strength Checker

A component that checks and displays the strength of a password in real-time, with customizable validation rules and visual feedback.

Basic Password Checker

Password Checker with Custom Rules

Custom Styled Password Checker

 
<!-- Basic Password Checker -->
<password-strength-checker></password-strength-checker>

<!-- Password Checker with Custom Rules -->
<password-strength-checker 
    min-length="8"
    require-numbers="true"
    require-special-chars="true"
    require-uppercase="true">
</password-strength-checker>

<!-- Custom Styled Password Checker -->
<password-strength-checker class="custom-checker"></password-strength-checker>
Property Type Default Description
min-length number 6 Minimum password length required
require-numbers boolean false Whether numbers are required
require-special-chars boolean false Whether special characters are required
require-uppercase boolean false Whether uppercase letters are required
Method Parameters Description
validate password: string Validates the given password and returns strength score
reset none Resets the checker to its initial state
CSS Variable Default Description
--password-checker-bg var(--color-card-bg) Background color of the checker
--password-checker-text var(--color-text) Text color of the checker
--password-checker-weak var(--color-error) Color for weak password strength
--password-checker-medium var(--color-warning) Color for medium password strength
--password-checker-strong var(--color-success) Color for strong password strength
--password-checker-padding 1rem Padding inside the checker
--password-checker-border-radius 4px Border radius of the checker
--password-checker-bar-height 4px Height of the strength indicator bar
--password-checker-bar-radius 2px Border radius of the strength indicator bar
--password-checker-transition 0.2s ease Transition animation for strength changes
