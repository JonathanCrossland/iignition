Editor MD Component

A markdown editor component with syntax highlighting, keyboard shortcuts, and markdown-specific features. Perfect for writing and editing markdown content with a clean, distraction-free interface.

Supported Markdown Syntax

  • Headers: # ## ### #### ##### ######
  • Emphasis: *italic* **bold**
  • Lists: - * + for unordered, 1. 2. 3. for ordered
  • Links: [text](url)
  • Images: ![alt](url)
  • Code: `inline code` and ```code blocks```
  • Blockquotes: > quoted text
  • Horizontal Rules: --- or ***
  • Tables: | column | column |

Smart Features

  • Automatic list continuation on Enter
  • Tab indentation for code and lists
  • Paste handling (strips HTML, preserves plain text)
  • Auto-save functionality
  • Keyboard shortcuts for common formatting
