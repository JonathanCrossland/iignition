Features

If you are after something really small with a few features, you could use this.

  • Data: Get JSON from .json file or a URL that returns JSONp
  • View: Load HTML files into a container
  • View: Uses #! in url
  • View: Splash the data onto some HTML (templating)
  • View: Splash with rowbind event
  • Cache: Uses LocalStorage
  • Utility: Switch off Console.Log
