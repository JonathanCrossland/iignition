Typewriter Component

A typewriter effect component that types out text character by character with optional blinking cursor.

<typewriter-component content="Your text here" cursor="true" tag="span" speed="50"></typewriter-component>
Property Type Default Description
content string "" The text to type out character by character
cursor boolean true Whether to display the blinking cursor
tag string span The HTML tag to wrap the content (h1,h2,h3,h4,h5,h6,span,p)
speed number 50 Typing speed in milliseconds between characters
Method Parameters Description
setContent(content) content: string Sets new content and starts typing
startTyping() none Starts or resumes the typing animation
stopTyping() none Stops the typing animation
reset() none Resets to beginning of text
complete() none Instantly completes the typing animation
Event Detail Description
typewriter-progress { text, progress, index } Fired on each character typed
typewriter-complete { text } Fired when typing animation completes
CSS Variable Default Description
--typewriter-font-family inherit Font family for the typewriter text
--typewriter-font-size inherit Font size for the typewriter text
--typewriter-color inherit Text color for the typewriter
--typewriter-line-height inherit Line height for the typewriter text
--typewriter-font-weight inherit Font weight for the typewriter text
--typewriter-cursor-color currentColor Color of the blinking cursor
--typewriter-cursor-speed 1s Speed of cursor blinking animation
--typewriter-cursor-weight normal Font weight of the cursor
--typewriter-cursor-opacity 1 Opacity of the cursor when visible
