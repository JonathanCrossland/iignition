AMMERSE Set Component

A visual component for displaying AMMERSE (Agile, Minimal, Maintainable, Environmental, Reachable, Solvable, Extensible) weight values in a grid layout.

Basic AMMERSE Set

AMMERSE Set without Title

Default Values

 
<!-- Basic usage with title and values -->
<ammerse-set title="My Weights" values="0.5,1,1,1,1,1,0"></ammerse-set>

<!-- Without title -->
<ammerse-set values="0.8,0.6,0.9,0.7,0.5,0.8,0.3"></ammerse-set>

<!-- Default values (all zeros) -->
<ammerse-set title="Default"></ammerse-set>
Property Type Default Description
title string none Optional title displayed above the AMMERSE grid
values string "0,0,0,0,0,0,0" Comma-separated values for A, Mi, M, E, R, S, Ex weights (range: -1 to 1)
Position Symbol Name Description
1 A Agile Flexibility and adaptability to changing requirements
2 Mi Minimal Simplicity and avoidance of unnecessary complexity
3 M Maintainable Clean, readable, and well-structured code
4 E Environmental Adherence to coding standards and best practices
5 R Reachable Feasible and achievable within project constraints
6 S Solvable Directly addresses the specific problem or requirement
7 Ex Extensible Designed for future modifications and additions
