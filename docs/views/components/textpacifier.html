Text Pacifier Component

A simple text-based loading indicator that displays a spinning character animation.





 
<text-pacifier 
    visible="true" 
    speed="200" 
    text="Loading...">
</text-pacifier>

Programmatic Usage

// Get the component
const pacifier = document.querySelector('text-pacifier');

// Show/hide the pacifier
pacifier.visible = true;  // Show
pacifier.visible = false; // Hide

// Change speed (in milliseconds)
pacifier.speed = 100; // Faster
pacifier.speed = 500; // Slower

// Change text
pacifier.text = 'Processing...';
Property Type Description
visible boolean Controls whether the spinner is active
speed number Rotation speed in milliseconds (default: 200)
text string The text to display next to the spinner (default: "Loading")
CSS Variable Default Description
--pacifier-font-size 1rem Font size of the pacifier text
--pacifier-color #333 Text color
--pacifier-bg transparent Background color
--pacifier-padding 0.5rem Padding around the component
--pacifier-radius 4px Border radius
